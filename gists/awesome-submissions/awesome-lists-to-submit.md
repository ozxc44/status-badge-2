## Submission Tips

1. **Search first** - Make sure the project isn't already listed
2. **Follow format** - Match the existing list's formatting style
3. **Keep it brief** - One line description is usually enough
4. **Add category** - Suggest where it fits alphabetically
5. **Be patient** - Maintainers are volunteers, follow up after 2-3 weeks
